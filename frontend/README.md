# Daily Report App

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. The app will run at [http://localhost:3000](http://localhost:3000).

## Environment

- Configure API base URL in `.env`:
  ```
  REACT_APP_API_BASE_URL=http://localhost:5000/api
  ```
